# **Restaurant Page**

## **Description**

A restaurant page built with Webpack.

From The Odin Project's [curriculum](https://www.theodinproject.com/lessons/node-path-javascript-restaurant-page).


## Live Demo
[Here](https://badger-cmyk.github.io/restaurant-page/)

## Built With
- HTML
- CSS
- JavaScript
- Webpack

## What I Learned
- How to set up Webpack to bundle web files
- How to configure Wepack loaders to handle different file types including CSS, images and fonts 
- How to work with ES6 modules and separate different concerns into different files
- How to dynamically manipulate the DOM
- How to use event delegation so events trickle down to children nodes from parent nodes
- How to write tab switching logic and enable tabbed browsing
- How to deploy a Webpack project to GitHub Pages
